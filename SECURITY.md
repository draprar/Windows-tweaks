# Security Policy

## Status

This repository is archived (read-only) and is not actively maintained.

## Reporting a vulnerability

If you believe a tweak in this repository introduces a security risk or unsafe behavior, you can:

- Open an issue, if issues are enabled, and include the exact file name plus your Windows version/build.
- Contact the repository owner via their GitHub profile if issues are unavailable.

Please note that, due to the archived status, responses are not guaranteed.

## Usage warning

Registry files (`.reg`) can change system behavior and security settings. Always review the contents before importing them, and preferably create a restore point or backup first.
